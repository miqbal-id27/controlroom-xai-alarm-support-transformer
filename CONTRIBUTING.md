# Contributing

Thanks for your interest! Please:
1. Create a feature branch from `main`.
2. Use `conda activate cr-xai`; run `ruff check .` and `black --check .`.
3. Add/adjust unit tests in `tests/`.
4. Ensure `pytest -q` passes locally.
5. Open a PR with a clear summary and link to a related Issue.

Security & data: do not upload proprietary control-room data. Use synthetic or public datasets.
